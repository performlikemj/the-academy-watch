# The Academy Watch — Documentation

Living, human-facing guides for the platform. Keep these current: when a feature
changes, update its guide **and** add a line to that guide's Changelog.

> These are **user/operator manuals**. Internal project state, decisions, and task
> tracking live in [`../ledgers/`](../ledgers) (the `CONTINUITY_*.md` files); deep
> technical research lives in [`../ledgers/research/`](../ledgers/research).
